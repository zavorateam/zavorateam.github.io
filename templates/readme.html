ZavoraTeam Website

Набор онлайн инструментов и полезных ресурсов для разработки сайтов и приложений

Структура проекта

  • /templates — HTML-шаблоны и страницы всех инструментов
  • /common_styles.css — Общие CSS-стили
  • /index.html — Файл перенаправления на основную страницу
  • /index_files — Ресурсы для главной страницы
  • /mod_files — Ресурсы и стили для BezmatMOD ZavoraMOD
  • /bible_files — Ресурсы для Paranoid's Bible RUS
  • /accums_files — Ресурсы для страницы аккумуляторов
  • /caleidoskopus_files — Файлы для визуализатора звука
  • /electrichki_files — Ресурсы для страницы электричек
  • /fonts — Шрифты

Доступные инструменты

1. Цветовые инструменты

Генератор палитр, конструктор CSS-градиентов, экспорт CSS-кода

2. CSS-анимации

Визуальный редактор анимаций, предпросмотр, генерация кода

3. Проверка битых ссылок

Массовая проверка URL, отчёты, визуализация статусов

4. Генератор favicon

Создание иконок для сайтов и PWA, генерация manifest и HTML-кода

5. Калейдоскопус

Визуализатор звука, экспорт SVG/PNG, поддержка MP3/WAV/FLAC

6. Табло электричек

Интерактивное табло с электричками

  • Обновление в реальном времени
  • Подгрузка из сохранённых файлов (Static.NoApi)
  • Подгрузка из Api ЦППК/РЖД/ФПК (Dynamic.Server)

7. Коды стран

Справочник телефонных и ISO-кодов

8. Текстовод

Инструменты для работы с текстом

9. Файловый менеджер

Загрузка, просмотр, группировка файлов, коллекции, drag-and-drop, темизация

Технологии

  • HTML5, CSS3, JavaScript (ES6+)
  • CSS Variables для темизации
  • LocalStorage для пользовательских настроек
  • Web Audio API, Canvas, SVG для визуализации
  • GitHub Pages для хостинга статических страниц
  • JWT для аутентификации
  • Drag-and-drop, адаптивный дизайн
  • Возможность расширения серверной части (Python/Go)

Особенности проекта

  • Темная/светлая тема — быстрое переключение, сохранение в LocalStorage
  • Адаптивный дизайн — поддержка мобильных и десктопных устройств
  • Офлайн-инструменты — большинство функций работают без сервера
  • Минималистичный и интуитивный интерфейс
  • Модульная структура — легко добавлять новые инструменты
  • Кроссбраузерность — поддержка всех современных браузеров
  • Работа с коллекциями файлов — группировка, предпросмотр, drag-and-drop
  • Интеграция Python/Go-скриптов — запуск и управление кодом локально

Разработка

  1. Клонируйте репозиторий: git clone https://github.com/zavorateam/zavorateam.github.io.git
  2. Перейдите в папку проекта: cd zavorateam.github.io
  3. Внесите изменения
  4. Отправьте изменения: git push или создайте pull request

Планы по развитию

  • Добавление новых инструментов для веб-разработчиков
  • Расширение Python или Go Explorer (API, коллекции, шаблоны)
  • Улучшение пользовательского интерфейса и темизации
  • Интеграция между инструментами (например, экспорт данных между модулями)
  • Расширение серверных функций (Flask, Go)
  • Расширение возможностей аудиовизуализации
  • Объединение Static и Dynamic версий табло электричек (без CORS)
  • Поддержка новых форматов файлов и коллекций
  • Доработка страниц (общие темы, собственный фреймворк)
