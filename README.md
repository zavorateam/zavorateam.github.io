## 🛠️ Доступные инструменты

### 🎨 Цветовые инструменты

Набор утилит для работы с цветами и градиентами:

- **Генератор палитр** — автоматическое создание гармоничных цветовых схем
- **CSS Gradient Builder** — конструктор градиентов с реальным предпросмотром
- **Экспорт кода** — готовый CSS для использования в проектах
- **Конвертация форматов** — HEX, RGB, HSL поддержка

### 🎬 CSS-анимации

Визуальный конструктор для создания сложных CSS-анимаций:

- Редактор ключевых кадров (keyframes)
- Настройка timing, duration, delay
- Предпросмотр в реальном времени
- Автогенерация CSS-кода

### 🔗 Проверка ссылок

Инструмент для валидации и проверки статуса URL:

- Массовая проверка (copy-paste список ссылок)
- Подробные отчеты о статусах
- Визуализация работающих/недоступных ссылок
- Экспорт результатов

### 🎯 Генератор favicon

Быстрое создание полного набора иконок для вашего сайта:

- Автогенерация размеров (16×16, 32×32, 64×64, 192×192, 512×512 и другие)
- Создание favicon.ico, PNG и WebP иконок
- Поддержка PWA (manifest.json)
- Готовый HTML-код для вставки в шапку

### 🎪 Калейдоскопус

Интерактивный визуализатор музыки и звука:

- Поддержка MP3, WAV, FLAC, OGG
- Множество режимов визуализации
- Экспорт в SVG и PNG
- Настройка цветов и параметров
- Real-time анализ спектра

### 🚂 Табло электричек

Интерактивное расписание московских электричек (РЖД, ЦППК, ФПК):

- **Static версия** — работает автономно (локальные данные)
- **Dynamic версия** — синхронизация с реальными API
- Две вкладки отправления (выбор Вперед/Назад)
- Дневное расписание по дням недели
- Быстрый поиск по станциям

### 🌍 Справочники

#### Коды стран
- ISO 3166-1 коды (2-х и 3-х буквенные)
- Телефонные префиксы
- Информация о странах и территориях

#### Информация о странах
- Столицы и города
- Население и площадь
- Временные зоны
- Валюты

### 📝 Текстовод

Набор текстовых утилит для обработки данных:

- Конвертация кодировок
- Форматирование текста
- Поиск и замена
- Статистика (символы, слова, строки)
- Удаление дубликатов и пробелов

### 📊 Временная шкала (Timeline)

Визуализация исторических событий и фактов:

- Документальные события
- Вымышленные события
- Интерактивная навигация
- Группировка по периодам

### 🎪 Прочие инструменты

- **Матрица** — цифровой дождь/матрица на canvas
- **Частицы** — демонстрация частичных систем
- **CSS Animations** — галерея анимаций
- **Планировщик** — органайзер и заметки
- **Markdown Editor** — редактор с предпросмотром
- **Matrix** — визуальные эффекты
- **CheMatter** — таблица химических элементов

## 👨‍💻 Разработка и вклад

### Запуск локального сервера

```bash
# На Python 3
python -m http.server 8000

# На Node.js
npx http-server

# На PHP
php -S localhost:8000
```

### Создание нового инструмента

1. Создайте HTML-файл в `templates/`
2. Добавьте CSS в `assets/` или внутри HTML
3. Добавьте JavaScript в `assets/` или внутри HTML
4. Создайте ссылку на главной странице
5. Следуйте существующему стилевому и архитектурному руководству

### Вклад в проект

```bash
git clone https://github.com/zavorateam/zavorateam.github.io.git
cd zavorateam.github.io
# ... внесите изменения локально ...
# Вклад в проект возможен через обсуждение с авторами
```
